# Enterprise-multi-VLAN-network
Enterprise-grade multi-VLAN network with Inter-VLAN routing using Layer-3 switching.

A complete enterprise-grade multi-VLAN network project built using Cisco Layer-3 switching, Inter-VLAN routing, DHCP, trunking, access ports, and security configurations.
This project simulates how real corporate networks segment traffic using VLANs and route between them using SVI and L3 interfaces.
